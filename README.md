**Investigate a TMBD Movies Dataset**

* Applied a hypothesis test to reach a decision on whether an old or a new web page led to a better conversion rate.
* Applied supervised machine learning by using Python statsmodels and Python scikit-learn to perform a logistic regression on multiple variables to predict which page had a higher conversion rate.
